- Division selector chip: 36px tall (compact, lives in header).

## Interaction Contracts

### Attendance tap (D-04)

- Entire player row is the tap target (min 44px tall, full width).
- Tap toggles between `ausente` (default, card in `--card` bg) and `presente` (card background shifts to club green at 20% opacity, border becomes solid club green).
- No separate "mark present" button exists. The row itself is the action.
- Visual feedback is immediate and optimistic (no spinner on tap). Queue failure is surfaced only if sync fails later.
- Inactive players (PLY-06) appear at the bottom of the list with an "Inactivo" badge; they are still tappable for attendance.

### Bottom navigation (D-01, D-02)

- 5 tabs: Jugadores · Lista · Fixture · Admin · Estadística.
- Phase 1 active tabs: **Jugadores** and **Lista** only.
- Disabled tabs: Fixture, Admin, Estadística — rendered at 40% opacity, no active state, no navigation on tap. Tapping shows a brief tooltip "Disponible próximamente" (use Sonner toast, 2s, no action button).
- Active tab: icon + label in club green. Inactive tabs: icon + label in `--muted-foreground`.
- Tab bar background: `--sidebar` token. Border-top: `--sidebar-border`.

### Division selector (D-03)

- Lives in the header, right-aligned.
- Rendered as a compact chip (36px tall, pill shape, `--secondary` background).
- On tap: opens a dropdown sheet (shadcn `Popover` or `Select`) listing assigned divisions.
- If coach has exactly one division assigned: chip is not rendered at all.
- Selected division persists in React Context + localStorage key `vrc_active_division`.

### Player avatar (D-06, D-07)

- In player list: circular avatar, 40px diameter. Fallback: initials on `--muted` background.
- In player profile: circular avatar, 96px diameter. Tap to upload new photo.
- Upload: opens native file picker (no drag-and-drop in Phase 1). Accepts image/* only.
- Client-side resize before upload: max 800×800px, JPEG quality 0.85, using `canvas.toBlob()` (no library). Upload to Supabase Storage.

### Offline indicator

- Shown as a persistent banner beneath the header when `navigator.onLine === false`.
- Background: `--muted` with amber text (`oklch(0.78 0.17 75)`). Single line: "Sin conexión — tomando lista en modo offline".
- Disappears immediately when connectivity is restored. On restore, sync queue runs; if successful, Sonner toast "Lista sincronizada" appears (2s, bottom of screen above nav).

### Disabled/placeholder screens

- Fixture, Admin, Estadística tabs navigate to a placeholder screen with centered icon (Lucide `Construction`, 48px, `--muted-foreground`) and text "Esta sección estará disponible próximamente." in `--muted-foreground` at 16px/400.
